# Върколак PDF implementation status

Source of truth: `C:\Users\Administrator\Desktop\върколаци-голяма-кутия-правила.pdf`.

Статуси:
- `implemented` означава, че ролята/правилото има runtime механика и regression покритие или съществуващ contract test.
- `mvp-basic` означава, че има работеща дигитална механика, но не е напълно 1:1 с PDF edge cases.
- `manual-only` означава, че ролята е в registry за ръчно водене/Разказвач, но default preset не я назначава.
- `planned` означава, че UI/типовете пазят място, но feature-ът остава скрит до wire-ване.

## Затваряне на edge-case audit

Към този статус файл няма останали `planned` или `mvp-basic` PDF правила за автоматичния дигитален Върколак loop. Остава само продуктово решение, не runtime липса: Крадец и Малко момиче се пазят извън default presets, защото изискват физически/допълнителни карти или ръчно водене, а не чисто автоматично поведение.
