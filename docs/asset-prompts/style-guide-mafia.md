# Мафия style guide

Visual direction: Sofia 90s crime-noir, rain, concrete blocks, smoky cafés, police folders, chess tables, neon reflected in asphalt.

Palette:
- Wet asphalt: `#142c2f`
- Rust red: `#7b1f1b`
- Old dossier paper: `#efe0bd`
- Ochre streetlight: `#c89a55`
- Dirty fluorescent yellow: `#e0aa63`

Setting cues:
- Sofia-inspired apartment blocks, old cafés, municipal corridors, pawn-shop windows, rain on tram tracks, cork-board evidence walls.
- Keep it Bulgarian/urban, not Italian mob cinema.

Style:
- Noir illustration with painterly texture, polaroid/dossier feeling, deep shadows, single overhead lamp.
- Faces feel tired, human and suspicious; no glamour poster poses.

Avoid:
- Fedoras, tommy guns, Godfather references, Las Vegas neon, clean luxury interiors, readable brand names, realistic celebrity likenesses, watermarks.

Prompt base:
```text
Painterly Sofia 90s crime-noir illustration for a Bulgarian Mafia social deduction game, wet asphalt and old dossier palette, single overhead lamp, deep shadows, rain reflections, evidence-board mood, no text, no watermark, no logo, no recognizable real person, no Italian mafia clichés, no photorealism.
```
