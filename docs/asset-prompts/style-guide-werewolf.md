# Върколак style guide

Visual direction: Balkan folk-horror, mountain village, suspicion around a table, moonlight through mist.

Palette:
- Deep night: `#0b1114`
- Old paper: `#f4e8d1`
- Dried blood: `#9e342d`
- Brass candlelight: `#d19a42`
- Fog: `#efe4ce`

Setting cues:
- Bulgarian/Balkan mountain village, stone walls, carved wood, wool, embroidered dividers, ritual masks used as background objects.
- Candlelight, moon haze, pine silhouettes, wet cobblestone, old table maps.

Style:
- Painterly illustration, not photo-real.
- Dramatic but grounded faces, warm key light, cool blue-green shadows.
- Quiet dread, not Halloween comedy.

Avoid:
- Generic medieval fantasy, US Halloween pumpkins, Twilight-style romance, furry mascot wolves, recognizable actors, readable text, watermarks.

Prompt base:
```text
Painterly Balkan folk-horror illustration for a Bulgarian Werewolf social deduction game, deep night palette, warm candle key light, cool fog shadows, textured old paper atmosphere, cinematic composition, no text, no watermark, no logo, no recognizable real person, no photorealism.
```
